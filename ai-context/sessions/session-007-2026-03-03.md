## Key Decisions / Content

### Storage Pool Naming
- Pools named by tier/purpose: `data`, `archive`, `backup`
- Datasets: `<pool>/<service>` — e.g. `data/nextcloud`, `archive/projects`
- Share names match dataset names for consistency

### Drive Testing Workflow (TrueNAS SCALE / zsh)
- TrueNAS uses zsh — must use array syntax: `DRIVES=(sda sdb sdc sdd sde sdf sdg)`
- `DRIVES="..."` quoted string fails in zsh for loops (no word splitting by default)
- Workflow: device mapping → health check → short tests → long tests → SMART data collection → Google Sheets
- Verdict criteria: PASS (<40k hours, zero errors), MONITOR (1-2 reallocated, 40-50k hours), REJECT (pending/offline errors, failed tests, >60k hours)

### Hardware Notes
- Crucial T705 2TB is double-sided — no rubber spacer needed on ASUS Hyper M.2 card
- Seasonic PSU hybrid mode: fan facing up (toward motherboard) per Seasonic recommendation
- Front drive cages preferred over bottom for thermals and noise in Define 7 XL
- ASUS Hyper M.2 x16 Gen 5 uses PCIe 6-pin power (use 6+2 cable, plug 6-pin portion)
